# 동역학 풀이판단 훈련기 v4

이 앱은 동역학 문제의 정답만 자동으로 내는 계산기가 아니라, 문제를 보고 어떤 풀이법으로 접근해야 하는지 훈련하는 웹앱입니다.

## v4 핵심 기능

1. **마법사형 단계별 UI**
   - 한 화면에 한 단계만 보여줍니다.
   - 문제 입력 → 구할 값 → 볼 물체/FBD → 좌표축 → 사용할 원리 → 식/단위 → 진단 순서로 진행합니다.
   - `st.session_state`로 현재 단계와 입력값을 유지합니다.

2. **문제 단서 하이라이팅**
   - 문제 문장 안에서 시간, 거리, 높이, 마찰 없음, 원형 트랙, 회전, 충돌 같은 단서를 형광펜처럼 표시합니다.
   - 학생이 “문제의 어떤 표현을 보고 풀이법을 떠올려야 하는지”를 직접 볼 수 있습니다.

3. **FBD 스케치 훈련**
   - `streamlit-drawable-canvas`를 이용해 웹에서 직접 힘 방향을 그릴 수 있습니다.
   - 문제 단서에 따라 중력, 수직항력, 장력, 마찰, 반지름 방향, 좌표축 등 FBD에 들어갈 가능성이 큰 항목을 체크합니다.

4. **풀이 방향 진단**
   - 규칙 엔진으로 운동학, F=ma, 일-에너지, 충격량-운동량, 원운동, 강체 평면운동, 복합 풀이 중 추천 풀이법을 제시합니다.
   - 풀이법 점수표, 판단 이유 로그, 문제 유형별 체크리스트를 제공합니다.

5. **학습 기록과 취약점 시각화**
   - 진단 결과를 오답노트처럼 저장할 수 있습니다.
   - 자주 빠지는 요소와 추천 풀이법 분포를 그래프로 보여줍니다.
   - 기록은 세션 기준이며, JSON 다운로드/불러오기를 지원합니다.

6. **선택적 AI 튜터 피드백**
   - API 키가 없어도 규칙 기반 진단은 작동합니다.
   - OpenAI API Key를 넣으면 학생 풀이에 대해 추가 피드백을 받을 수 있습니다.

## 이전 버전에서 이어진 개선점

- `마찰 없는`, `frictionless`, `no friction` 같은 부정 표현을 처리합니다.
- `위`, `아래`, `반지름`, `N`처럼 너무 넓어서 오진을 만들던 키워드를 보수적으로 바꿨습니다.
- 원형 트랙의 반지름과 원판/바퀴 자체의 반지름을 구분합니다.
- 포물선 문제는 최고 높이가 있어도 기본적으로 운동학 풀이를 우선 추천합니다.
- 경사면 가속도 문제는 불필요하게 복합 풀이로 넘어가지 않도록 조정했습니다.
- 등가속도 계산기는 입력값끼리 모순되는 후보해를 제외하고 경고를 보여줍니다.
- 회귀 테스트를 통해 대표 예제가 계속 의도대로 분류되는지 확인할 수 있습니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

규칙 엔진만 빠르게 확인하려면 다음을 실행합니다.

```bash
python regression_tests.py
```

## 웹 배포 방법

1. GitHub 저장소를 만듭니다.
2. 이 폴더 안의 파일을 업로드합니다.
3. Streamlit Community Cloud에서 저장소를 연결합니다.
4. main file path를 `app.py`로 지정하고 Deploy합니다.

## 선택적 AI 피드백

API 키가 없어도 규칙 기반 진단은 작동합니다.
AI 피드백을 쓰려면 앱 사이드바에 OpenAI API Key를 입력하거나, Streamlit Cloud의 Secrets에 다음처럼 저장합니다.

```toml
OPENAI_API_KEY = "sk-..."
```

## 앱의 의도

이 앱은 정답을 바로 주기보다 아래 질문을 반복 훈련하게 만듭니다.

- 이 문제는 무엇을 구하는가?
- 주어진 단서는 무엇인가?
- 어떤 물체를 따로 떼어 볼 것인가?
- FBD에는 어떤 힘을 그려야 하는가?
- 좌표축과 양의 방향은 어떻게 잡을 것인가?
- 시간과 연결되는가, 거리/높이와 연결되는가?
- 한 공식으로 끝나는가, 여러 원리를 연결해야 하는가?

## 주의

규칙 기반 진단은 학습 보조용입니다. 복잡한 강체 동역학 문제나 그림 의존도가 큰 문제는 실제 해설, 교수님 풀이, 교재 풀이와 함께 확인하는 것이 좋습니다.
