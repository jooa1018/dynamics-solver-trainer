# 동역학 풀이판단 훈련기 v3

이 앱은 동역학 문제의 정답만 자동으로 내는 계산기가 아니라, 문제를 보고 어떤 풀이법으로 접근해야 하는지 훈련하는 웹앱입니다.

## v3 핵심 기능

1. 문제 문장 입력
2. 학생 풀이 입력
3. 풀이법 선택 후 피드백
4. 문제 단서 자동 감지
5. 직접 단서 체크
6. 복합 풀이 감지
7. 빠진 풀이 요소 진단
8. 오개념 패턴 탐지
9. 판단 이유 로그
10. 문제 유형별 필수 체크리스트
11. 계산 도우미
12. 선택적 AI 튜터 피드백


## 이번 수정본에서 개선된 점

- `마찰 없는`, `frictionless`, `no friction` 같은 부정 표현을 처리합니다.
- `위`, `아래`, `반지름`, `N`처럼 너무 넓어서 오진을 만들던 키워드를 보수적으로 바꿨습니다.
- 원형 트랙의 반지름과 원판/바퀴 자체의 반지름을 구분합니다.
- 포물선 문제는 최고 높이가 있어도 기본적으로 운동학 풀이를 우선 추천합니다.
- 경사면 가속도 문제는 불필요하게 복합 풀이로 넘어가지 않도록 조정했습니다.
- 등가속도 계산기는 입력값끼리 모순되는 후보해를 제외하고 경고를 보여줍니다.
- 단계별 풀이 입력칸을 추가해 문제 해석, 물체 선택, 좌표축, 원리, 식, 단위 확인을 나눠 점검할 수 있습니다.
- `regression_tests.py`를 추가해 대표 예제가 계속 잘 분류되는지 확인할 수 있습니다.
- 추천 풀이법이 왜 나왔는지 추적할 수 있는 **판단 이유 로그**를 추가했습니다.
- 운동학, F=ma, 일-에너지, 충격량-운동량, 원운동, 강체/굴림별 **필수 체크리스트**를 추가했습니다.
- 회귀 테스트를 확장해 경사면, 스프링, 도르래, 원판/원형 트랙 구분, 거리 감지, 체크리스트 생성까지 확인합니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

규칙 엔진만 빠르게 확인하려면 다음을 실행합니다.

```bash
python regression_tests.py
```

## 웹 배포 방법

1. GitHub 저장소를 만듭니다.
2. 이 폴더 안의 파일을 업로드합니다.
3. Streamlit Community Cloud에서 저장소를 연결합니다.
4. main file path를 `app.py`로 지정하고 Deploy합니다.

## 선택적 AI 피드백

API 키가 없어도 규칙 기반 진단은 작동합니다.
AI 피드백을 쓰려면 앱 사이드바에 OpenAI API Key를 입력하거나, Streamlit Cloud의 Secrets에 다음처럼 저장합니다.

```toml
OPENAI_API_KEY = "sk-..."
```

## 앱의 의도

이 앱은 정답을 바로 주기보다 아래 질문을 반복 훈련하게 만듭니다.

- 이 문제는 무엇을 구하는가?
- 주어진 단서는 무엇인가?
- 시간과 연결되는가, 거리/높이와 연결되는가?
- 힘을 직접 묻는가?
- 충돌인가?
- 회전인가?
- 한 공식으로 끝나는가, 여러 원리를 연결해야 하는가?

## 주의

규칙 기반 진단은 학습 보조용입니다. 복잡한 강체 동역학 문제나 그림 의존도가 큰 문제는 실제 해설, 교수님 풀이, 교재 풀이와 함께 확인하는 것이 좋습니다.
