# 바이브코딩 확장 프롬프트

## 1. 문제 이미지 OCR 추가

```text
현재 Streamlit 앱에 문제 이미지 업로드 기능을 추가하고 싶다.
사용자가 이미지를 업로드하면 OCR로 문제 문장을 추출하고, 기존 v3 풀이 진단 입력창에 자동으로 넣어줘.
단, OCR 결과가 틀릴 수 있으니 사용자가 수정할 수 있게 해줘.
```

## 2. 단계별 풀이 진단 추가

```text
학생 풀이를 한 번에 입력하는 대신, 1단계 문제 해석, 2단계 FBD/좌표축, 3단계 공식 선택, 4단계 식 세우기, 5단계 계산으로 나누어 입력하게 만들고 싶다.
각 단계마다 피드백을 주도록 Streamlit UI와 진단 함수를 수정해줘.
```

## 3. 오답노트 저장 기능

```text
Streamlit 앱에 오답노트 저장 기능을 추가하고 싶다.
문제, 학생 풀이, 추천 풀이법, 빠진 요소, 오개념을 JSON 파일로 저장하고, 앱 안에서 다시 볼 수 있게 해줘.
```

## 4. 문제 유형별 공식 카드 확장

```text
동역학 풀이판단 앱의 오개념 노트 탭에 공식 카드를 추가하고 싶다.
운동학, F=ma, 일-에너지, 충격량-운동량, 원운동, 강체 평면운동별로 핵심 공식과 적용 조건, 주의점을 표로 정리해줘.
```
